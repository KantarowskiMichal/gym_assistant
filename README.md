# Gym Assistant

A Flutter app for planning and scheduling workouts.

## Features

- **Exercises**: Define custom exercises with default sets, reps/duration, and weight
- **Workouts**: Create workout templates from collections of exercises
- **Calendar**: Schedule workouts with recurrence (one-off, weekly, or every N days)

## Getting Started

### Prerequisites

- Flutter SDK (^3.10.1)
- Android Studio / VS Code with Flutter extension

### Commands

```bash
flutter pub get          # Install dependencies
flutter run              # Run the app
flutter test             # Run tests
flutter analyze          # Static analysis
```
